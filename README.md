# CARL Air Brake Motherboard Software

This repository contains the software for the motherboard of the
[CARL Air Brake][cab-repo] computer. The CARL Air Brake computer automatically
controls a rocket air brake in flight, so that the vehicle attains an apogee at
the desired altitude. Its motherboard (Raspberry Pi 3 model A+) is responsible
for executing the control and state estimation algorithms, and whatever other
tasks are required in flight.

**Note**: This repository is currently work-in-progress. The CARL Air Brake was
already flown once with an older version of this software. Currently, the
software is being restructured and improved, and will then be available on this
repository.
